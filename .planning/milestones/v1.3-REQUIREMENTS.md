# Requirements: Helix v1.3

**Defined:** 2026-02-12
**Core Value:** Group exercise management for live coaching with reusable templates

## v1.3 Requirements

Requirements for milestone v1.3 Image Auto-Play.

### Image Auto-Play

- [x] **IMGAP-01**: Coach can tap an exercise image to start automatic slideshow (3 sec interval, loops)
- [x] **IMGAP-02**: Coach can tap again to stop the slideshow
- [x] **IMGAP-03**: Play/pause icon overlay is visible on the image indicating current state
- [x] **IMGAP-04**: Manual swipe stops auto-play if active
- [x] **IMGAP-05**: Auto-play is only available for exercises with multiple Lumio images (single-image exercises are unaffected)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Configurable slideshow speed | 3 sec fixed is sufficient for coaching context |
| Auto-play on main app (non-live) | Feature is tablet live-only |
| Video/animation support in Lumio cards | Out of scope, images only |
